# Car Plate Detector

Car Plate Detector is an Android app that detects vehicle license plates in the camera preview and highlights them with a bounding box. A beep can be played when a plate is detected.

> Note: The current version performs **plate detection only**. OCR (reading plate text) is planned for a future update.

## Features
- Real-time license plate detection (on-device)
- Bounding box overlay
- Optional beep alert on detection
